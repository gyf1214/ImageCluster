# ImageCluster
## Homework of DSA Practice

初始文件包含  
*main.cc*  
*util.cc*  
*util.hpp*  
*vector.cc*  
*vector.hpp*  

使用GNU make编译,直接通过命令make即可  
之后会生成  
*main.o*  
*util.o*  
*vector.o*  
**main.exe**  
其中**main.exe**为该可执行文件  
用命令行执行main.exe时  
命令行参数中第一个为输入文件名，第二个为输出文件名  
默认为test.txt与result.txt 并生成在其目录下
